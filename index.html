Introduction à HTML

1. Définition

HTML (HyperText Markup Language) est le langage de base utilisé pour créer des pages web. Il structure le contenu en utilisant des balises.

1. Structure de base d'une page HTML

!DOCTYPE html : Indique au navigateur que le document est en HTML5.
HTML : Conteneur principal de la page.
Head : Contient les métadonnées, comme le titre et les liens vers les fichiers CSS.
Body : Contient tout le contenu visible de la page.
